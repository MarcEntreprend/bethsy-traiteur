# Bethsy Saveur de l'Île — Showcase Website

A refined, high-end showcase website crafted for **Bethsy Saveur de l'Île**, an exceptional catering and fine-dining service based in Cap-Haïtien, Haiti. The platform offers an immersive visual experience that highlights traditional Creole culinary arts blended with modern elegance.

> **Description:** A refined showcase website for "Bethsy Saveur de l'Île". An immersive, high-end experience crafted with modern HTML/CSS, highlighting the island's culinary art and cultural authenticity through a sleek design, meticulous typography, and a seamless user interface.

---

## Features

- **Premium UI/UX Design:** Immersive luxury aesthetic using tailored typographic hierarchy (`Cormorant Garamond` and `Montserrat`).
- **Dynamic Navigation:** Fixed, blur-filtered (`backdrop-filter`) navigation bar responding smoothly to scroll transitions.
- **Visual Micro-Interactions:** Custom SVG vector branding enhancements with subtle drop-shadow filters ensuring premium contrast across dynamic light backgrounds.
- **Scroll-to-Reveal System:** Light-weight `IntersectionObserver` JavaScript implementation orchestrating elegant fade-up animations.
- **Fully Responsive Architecture:** Optimized layout shifting fluently across ultra-wide, standard desktop, tablet, and mobile displays.

---

## Tech Stack & Architecture

- **Markup:** Semantic HTML5 structure.
- **Styling:** Vanilla CSS3 utilizing native Custom Properties (`--variables`), standard structural Flexbox grids, and fluid font sizing (`clamp()`).
- **Interactions:** Modern, passive-listener JavaScript for efficient scroll performance monitoring.
- **Typography:** Google Fonts Integration (Serif: _Cormorant Garamond_ / Sans: _Montserrat_).
